# Mouse Gestures for IntelliJ IDEA

![Plugin Icon](src/main/resources/META-INF/pluginIcon.svg)

A simple and intuitive mouse gestures plugin for IntelliJ IDEA. Perform common IDE actions like 'Back', 'Forward', or 'Close Tab' simply by moving your mouse.

## 🚀 Features
- **Fast Navigation:** Speed up your workflow with predefined gestures.
- **Natural Interaction:** Smooth gesture recognition integrated directly into the editor.
- **Open Source:** Completely free for everyone under the MIT License.

## 🛠 Installation
1. Open IntelliJ IDEA.
2. Go to `Settings` (or `Preferences` on macOS) > `Plugins`.
3. Search for **Mouse Gestures** in the Marketplace.
   *OR*
   Download the latest ZIP/JAR from the [Releases](https://github.com/Aadammm/Mouse_Gestures/releases) page and install it via "Install Plugin from Disk...".

## 🖱 How to Use
Hold the **Right Mouse Button** (default) and draw the gesture on the screen:

| Gesture | Action |
| :--- | :--- |
| **←** (Left) | Navigate Back |
| **→** (Right) | Navigate Forward |
| **↓** (Down) | Close Current Tab |
| **∟** (Right then Up) | Create New File |
